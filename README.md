📌 Présentation
SecurePass est une application web permettant de générer des mots de passe forts et personnalisés directement dans le navigateur, sans aucune installation.
Projet réalisé en autodidacte dans le cadre de ma montée en compétences en développement web front-end.

✨ Fonctionnalités

🎚️ Longueur personnalisable de 8 à 32 caractères
☑️ Choix des types de caractères : majuscules, minuscules, chiffres, symboles
💪 Indicateur de force du mot de passe en temps réel
📋 Copie en un clic dans le presse-papiers
💡 Conseils de sécurité intégrés
📱 Interface responsive (mobile & desktop)


🛠️ Technologies utilisées
HTML5 -> Structure de la page, CSS3 -> Mise en forme, JavaScript (Vanilla) -> Logique de génération, DOM, événements  Bootstrap, Icônes Google Fonts (Raleway)Typographie
